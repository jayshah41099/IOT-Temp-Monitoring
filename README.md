# IOT-Temperature-Monitoring-Alert-Automation-System

A real-time IoT-based temperature monitoring system designed to detect anomalies and trigger automated alerts. 
This project integrates Node-RED workflows and custom validation scripts to process sensor data, ensuring reliability and accuracy.

Sensor data is stored in an InfluxDB time-series database for efficient querying, while Grafana dashboards provide intuitive visualization of both real-time and historical trends. 
The system enables proactive monitoring through threshold-based alerting, making it suitable for environments that require continuous temperature oversight.
